# Frontend Ley REP

Frontend construido con **Vue 3 + Vite** y **CSS nativo responsive**, sin Bootstrap.

## Desarrollo local
1. Instala dependencias:
   ```bash
   npm install
   ```
2. Inicia Vite:
   ```bash
   npm run dev
   ```
3. Abre `http://localhost:5173`.

El frontend usa proxy hacia `http://localhost:3000` para las rutas `/api`.
